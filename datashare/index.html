How Blocklith Datashare works

See how to transfer pre-indexed, enriched blockchain data from our environment to yours.

  1. Select the data

    Choose from prebuilt, curated crypto datasets like Stablecoin Transfers or DEX Trades; or define your own blockchain dataset using token, wallet, contract, and event filters.

  2. Select the format

    Choose your preferred export format for the data, e.g. JSON, Parquet, or CSV. Blocklith Datashare offers data formats optimised for data lakes, data warehouses & AI/ML pipelines.

  3. Select the destination

    Deliver the data directly to your cloud data platform. Choose from Amazon S3, Google BiqQuery, Snowflake, Kafka, Databricks and more - or send the data via API or direct file upload.

  4. Export the datasets

    Start the data pipeline with one-click exports. Schedule recurring data deliveries or run one-time exports with data snapshots. Blockchain data, live in your stack.
